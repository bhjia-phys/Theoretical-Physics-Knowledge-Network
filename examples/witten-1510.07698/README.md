# Witten Exemplar

This directory tracks the first exemplar for the kernel:

- source: Edward Witten, *Three Lectures On Topological Phases Of Matter*
- arXiv: `1510.07698v2`
- current scope: Lecture One only

## Why this source

Lecture One is a good seed corpus because it naturally couples:

- local band-crossing analysis;
- global topological constraints;
- Berry-connection geometry;
- explicit lattice toy models;
- bulk-boundary intuition.

## Intended Queries

The current object set is designed to answer queries like:

- what makes a 3D isolated two-band crossing a Weyl node;
- why net chirality on a lattice must vanish;
- how Berry curvature detects node charge;
- why Fermi arcs are tied to projected bulk Weyl nodes;
- what extra assumptions are needed before calling a crossing a Weyl semimetal.
