# Jackiw-Rebbi Zero Mode for a Sign-Changing Dirac Mass

- ID: `theorem:jackiw-rebbi-zero-mode-for-sign-changing-dirac-mass`

## Summary
If a Dirac mass profile changes sign across an interface, the Dirac operator admits a normalizable bound state whose envelope is the exponential of minus the mass integral.

## Scope
Reusable theorem behind the domain-wall derivation of chiral boundary modes in Lecture Three.

## Regime
Continuum Dirac operator with a one-dimensional sign-changing mass profile.

## Assumptions
- `The asymptotic masses at x_1 -> plus/minus infinity have opposite sign.`
- `The tangential momentum is set to the zero-mode value when solving the transverse localization problem.`
